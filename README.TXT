Implementação do problema do cavalo[1] utilizando o ambiente do Greenfoot[2]. O problema consiste em implementar um algoritmo para fazer com que um Cavalo (do jogo de xadrez) 
percorra todas as casas do tabuleiro sem que ele passe novamente por uma casa previamente percorrida.
